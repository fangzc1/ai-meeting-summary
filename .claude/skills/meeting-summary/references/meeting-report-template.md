## 填充指南

### 通用规则
1. **语言**: 报告统一使用中文输出，即使原始会议中包含英文内容
2. **英文术语**: 保留行业标准英文术语原词（如 API、UI、Sprint）
3. **时间引用**: 使用 `HH:MM:SS` 格式引用视频中的时间点
4. **截图嵌入**: 使用 Markdown 图片语法 `![文件名](screenshots/文件名.jpg)` 嵌入截图，文件名为时间戳格式如 `00m25s-05m20s.jpg`

### 章节优先级
- **必填**: 会议概述、议题与结论、Action Items
- **有则填**: 决策记录、待议事项、参会者发言摘要
- **无内容时**: 标注 "本次会议未涉及此项" 而非留空

### 智能判断
- 如果转录中能识别不同说话人（通过称呼、语气转换），尽量区分
- 如果截图中包含 PPT/文档内容，在议题中引用截图说明
- Action Items 的优先级根据上下文语气判断："必须"/"紧急" = 高，"建议"/"可以" = 低
